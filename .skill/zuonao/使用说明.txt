🧠 左脑 — 使用说明 v3.12
========================

一、这是什么？
左脑 = 记忆鸭记忆引擎 + 输出熊猫推理引擎 + 感知增强系统 + KAR融合模块 + 自检系统。
采用加密狗1授权方案（一机一密专属授权）。

架构定位：左脑是用户级 Skill（物理位置） + 系统级内核（架构身份）。
安装后，全局 IDENTITY.md 自动注入到所有会话的 system prompt，
无需每次新会话手动操作——安装即内核。

包含六大核心能力：
  1. 记忆引擎：知识存储、图扩散检索、智能纠错、多义词消歧、SimHash语义匹配
  2. 推理引擎：数据分析（数字/趋势/对比）、文章总结、体裁分类、纠缠场关联、因果链提取
  3. 感知增强：自动感知意图、自动学习、上下文注入、智能推荐
  4. KAR融合：知识-关联-推理统一管线处理
  5. 自检系统：10项检查 + 7项自动修复
  6. Token监测：实时消耗/节省/费用/趋势/热力图

二、安装步骤
  1. 将收到的「左脑专属授权包_订单号.zip」解压到任意目录
  2. 安装到 WorkBuddy：将整个文件夹复制到 ~/.workbuddy/skills/
  3. 在 WorkBuddy 中执行：/启动左脑 授权校验
  4. 验证通过后执行：/左脑 session（自动初始化全局架构 + 加载记忆）
  5. 验证架构状态：/左脑 架构（确认全局 IDENTITY.md 已生效）
  6. 完成！之后所有新会话自动继承，无需重复操作

三、使用方式

  万能入口（推荐）：
    /左脑 <你想做的事>         自动识别8种意图：学习/查询/搜索/分析/总结/纠错/推理推荐/追溯

  核心命令：
    /左脑 session              会话初始化（加载记忆 + 更新注入文件）
    /左脑 架构                 查看架构注入状态（全局 IDENTITY.md 是否生效）
    /左脑 自检                 10项全量检查 + 7项自动修复
    /左脑 仪表盘               打开 Token 监测浮动窗口

  记忆管理：
    /左脑 修改 旧文本|新文本    修改已记错的知识
    /左脑 删除 <关键词>         删除一条知识
    /左脑 记忆列表 [页码]       分页浏览记忆
    /左脑 待确认               查看自动学习的知识（确认/忽略）
    /左脑 追溯 <关键词>         追溯知识来源和对话上下文

  Workspace 管理：
    /左脑 工作区               当前工作区 + 各工作区知识分布
    /左脑 工作区 global        切换到全局工作区
    /左脑 全局搜索 <关键词>     跨所有工作区搜索

  高级命令：
    /左脑 纠缠 <词>            纠缠场关联分析
    /左脑 意图 <文本>           查看意图分类结果
    /左脑 上下文               查看当前对话上下文栈
    /左脑 用户画像             查看使用画像
    /左脑 建议                 查看优化建议
    /左脑 导出 [类型]           导出 CSV

四、自检系统（10项检查 + 7项修复）

  | # | 检查项           | 修复能力                                   |
  |---|------------------|---------------------------------------------|
  | 1 | 数据库完整性      | 清理空/损坏节点，重建索引                   |
  | 2 | 注入一致性        | 重写 _inject.md，聚合全workspace数据       |
  | 3 | 对话历史可达      | 需AI调用conversation_search验证            |
  | 4 | Memory日志        | 检测日志目录/文件可读性                     |
  | 5 | 纠缠场一致性      | 清理断裂关联                                |
  | 6 | workspace隔离     | 检测归属异常                                |
  | 7 | 授权状态          | 检查license.lic有效性                       |
  | 8 | Token数据库       | JSON vs SQLite计数同步                      |
  | 9 | 备份时效          | 超过30天告警                                |
  |10 | 架构注入状态      | 检查全局IDENTITY.md→不存在则写入，缺少则追加|

  快速排查架构：/左脑 架构（仅检查第10项，秒级返回）

五、Token 监测助手（浮动窗口）
  想实时看着左脑帮你省了多少 Token？
  1. 完成授权验证后，双击「ToKen监测助手.py」
  2. 一个半透明浮动窗口会出现在屏幕右上角
  3. 自动每 3 秒刷新，实时显示：
     · 已节省 Token + 折合金额
     · 节省时间 + 日均增速
     · 等效生成速度（token/s）
     · 知识图谱节点数 + 上下文使用率
     · 各功能活跃度排行
     · CO₂ 减排量
  4. 可拖拽到任意位置，点 ✕ 关闭

六、注意事项
  - 本技能采用加密狗1授权方案，授权绑定你的电脑硬件指纹
  - 换电脑、修改授权文件均会失效
  - 禁止转发、共享、倒卖此技能
  - 如需续费或换绑，请联系卖家处理
  - 左脑数据全局共享（~/.workbuddy/skills/左脑/data/），workspace标签区分上下文归属

八、排查架构问题
  安装后遇到问题？按以下步骤排查：
  1. /左脑 架构 → 快速查看全局 IDENTITY.md 是否生效
  2. /左脑 自检 → 10项全量检查，自动修复已知问题
  3. 检查 ~/.workbuddy/IDENTITY.md 是否包含"左脑处理层"标记
  4. 检查 ~/.workbuddy/skills/左脑/data/memory_duck_data.json 是否存在

  架构未生效的常见原因：
  · 未执行 /左脑 session 初始化 → 执行一次即可
  · 全局 IDENTITY.md 被误删 → /左脑 session 会自动重建
  · 授权未激活 → 先执行 /启动左脑 授权校验

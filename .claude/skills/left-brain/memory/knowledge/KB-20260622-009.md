# KB-20260622-009

时区修复 - save.js 之前用 toISOString() 返回 UTC 时间（00:12），用户看是 08:17 本地时间。改为本地时间：用 getTimezoneOffset() 偏移修正。文件名前缀从 UTC 变成本地（08:19）。注意：之前生成的快照文件名仍是 UTC 时戳（00:12 等），不影响内容可读性。

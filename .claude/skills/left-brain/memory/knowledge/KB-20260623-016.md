# v1.9 快照模式可控化

## 核心改进
- `/snap-mode` 会话级切换：off / manual / milestone / auto / reset
- 优先级链：session-snap-mode.json（会话级） > snapshot-config.json（全局） > 默认 milestone
- 状态栏 `📸mstone` / `📌off` 实时可见
- 修复 milestone 关键词对显式 save.js 调用也生效

## 新增文件
- scripts/会话快照/snap-mode.js — 模式管理
- .claude/skills/left-brain/scripts/get-snap-mode.js — 共享解析
- .claude/commands/snap-mode.md — slash 命令
- .claude/commands/snap-save.md — 强制存一次

## 关键 bug 修复
原 save.js: `if (config.mode === 'milestone' && !isExplicitCall && !force)` — 显式调用绕过关键词检查
现: `if (config.mode === 'milestone' && !force)` — 显式调用也走关键词检查

## 验证
7/7 场景：off/手动/manual/auto 各场景符合预期；mode=milestone 普通标签跳过、完成标签保存

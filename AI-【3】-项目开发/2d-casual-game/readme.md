# 2D 休闲闯关小游戏

> 纯前端 HTML5 + Canvas 小游戏，双击即玩

## 运行方式

1. 双击 `index.html`
2. 浏览器自动打开游戏
3. 点击「开始游戏」
4. 用方向键 `↑↓←→` 控制角色移动

## 游戏规则

- ⭐ **收集星星**：碰到星星 +10 分
- 💀 **躲避障碍**：碰到红色方块 -1 生命
- ❤ 初始 3 条命，生命归零游戏结束
- ⏱ 总时长 90 秒，时间到自动结算

## 操作说明

| 按键 | 功能 |
|:-----|:-----|
| `↑` `W` | 向上移动 |
| `↓` `S` | 向下移动 |
| `←` `A` | 向左移动 |
| `→` `D` | 向右移动 |

支持多键同时按下（斜向移动）。

## 技术栈

- HTML5 Canvas
- 原生 JavaScript（无框架）
- 原生 CSS
- 无任何外部依赖

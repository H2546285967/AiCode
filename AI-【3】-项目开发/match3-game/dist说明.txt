======================================
三消消消乐 - 本地打包分发说明
======================================

【打包方式】

本游戏为纯前端单文件，无需任何构建步骤。
只需将 index.html 一个文件发送给对方即可运行。

【分发方式】

方式 1：直接发送 index.html 文件
  - 微信发送、邮件附件、U盘拷贝均可
  - 对方双击 index.html 即可玩

方式 2：打包为压缩包
  - 将以下 3 个文件一起打包：
    * index.html
    * readme.md
    * dist说明.txt
  - 压缩为 .zip 发送给对方
  - 解压后双击 index.html 运行

【运行环境】

  - 任意现代浏览器（Chrome / Edge / Firefox / Safari）
  - Windows / Mac / Linux 均可
  - 无需安装任何软件
  - 无需联网（完全离线运行）

【注意事项】

  1. 不要修改 index.html 的文件名
  2. 不需要搭建服务器
  3. 不需要安装 Node.js 或任何运行时
  4. 如果想修改游戏参数，用记事本打开 index.html
     修改顶部 CONFIG 配置区即可

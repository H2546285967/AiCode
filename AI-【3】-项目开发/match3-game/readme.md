# 三消消消乐小游戏

> 纯前端 HTML5 + Canvas 三消游戏，双击即玩

## 运行方式

1. 双击 `index.html`
2. 浏览器自动打开游戏
3. 点击「开始游戏」
4. 鼠标点击选中方块，再点击相邻方块交换

## 游戏规则

- 🎯 点击两个**相邻**方块交换位置
- ⭐ 横向或纵向 **3 个及以上相同颜色**连在一起自动消除
- 💯 得分：3 个 = +10 分，4 个 = +25 分，5 个+ = +50 分
- 🔗 连锁消除每多一层 +15 分
- 👣 初始 35 步，每交换一次消耗 1 步
- 无效交换（不形成三消）会自动撤回，不扣步数
- 🏆 步数归零游戏结束，显示最终得分

## 操作说明

| 操作 | 方式 |
|:-----|:-----|
| 选中方块 | 鼠标左键点击 |
| 交换方块 | 选中后点击相邻方块 |
| 取消选中 | 再次点击已选中的方块 |

## 参数修改

打开 `index.html`，找到顶部 `CONFIG` 配置区，可修改：

| 参数 | 默认值 | 说明 |
|:-----|:-------|:-----|
| `ROWS` | 8 | 棋盘行数 |
| `COLS` | 8 | 棋盘列数 |
| `BLOCK_TYPES` | 5 | 方块颜色种类数 |
| `INITIAL_STEPS` | 35 | 初始步数 |
| `SCORE_3` | 10 | 3 连消除得分 |
| `SCORE_4` | 25 | 4 连消除得分 |
| `SCORE_5` | 50 | 5+ 连消除得分 |
| `CHAIN_BONUS` | 15 | 每层连锁加分 |

## 技术栈

- HTML5 Canvas
- 原生 JavaScript（无框架）
- 原生 CSS
- 无任何外部依赖

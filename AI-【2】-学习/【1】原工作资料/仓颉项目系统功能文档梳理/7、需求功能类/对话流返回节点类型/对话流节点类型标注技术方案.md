# 对话流节点类型标注技术方案

## 📋 **文档概述**

本文档详细说明了如何在对话流的**回复节点**和**输出节点**返回数据时，增加 `nodeTypeName` 字段用于标注内容来源类型，方便前端进行差异化展示。

---

## 🎯 **需求背景**

### **问题描述**
对话流执行时，回复节点和输出节点会返回多段内容：
- 大模型生成的内容
- 代码节点/HTTP 节点的计算结果
- 全局变量/局部变量的值
- 节点配置的静态文本

前端无法区分每段内容的来源，无法进行差异化展示（如显示不同的图标、样式等）。

### **解决方案**
在 SSE 流式返回的每条数据中增加 `nodeTypeName` 字段，标识该段内容的来源类型。

---

## 📊 **节点类型分类规则**

### **类型定义**

| 类型名称       | 标识值          | 说明                   | 示例                                      |
| -------------- | --------------- | ---------------------- | ----------------------------------------- |
| **大模型节点** | `"llm-node"`    | 大模型生成的内容       | Qwen、GPT 等模型输出                      |
| **其他节点**   | `"other-node"`  | 非大模型的计算节点输出 | 代码节点、HTTP 请求、变量赋值、数学计算等 |
| **回复节点**   | `"reply-node"`  | 回复节点自身的静态文本 | 节点配置中直接编写的文本                  |
| **输出节点**   | `"output-node"` | 输出节点自身的静态文本 | 节点配置中直接编写的文本                  |

---

### **分类逻辑**

#### **1. 回复节点 (reply-node)**

**配置示例**:
```
回复内容：你好，${大模型_1 / #llmText}，这是${代码节点 / #result}
```


**返回数据**:
```javascript
data: {"context":"你好，", "nodeTypeName":"reply-node"}      // 静态文本
data: {"context":"大模型回答内容", "nodeTypeName":"llm-node"}  // 大模型变量
data: {"context":"，这是", "nodeTypeName":"reply-node"}      // 静态文本
data: {"context":"代码运行结果", "nodeTypeName":"other-node"} // 非大模型变量
```


**判断规则**:
- ✅ 静态文本 → `"reply-node"`
- ✅ 变量名以 `llmText` 结尾 → `"llm-node"`
- ✅ 其他变量 → `"other-node"`

---

#### **2. 输出节点 (output-node)**

**配置示例**:
```
输出：
  大模型_1 / #llmText
  全局变量 / #name
  全局变量 / #age
  22222222222222222222
```


**返回数据**:
```javascript
data: {"context":"大模型输出内容", "nodeTypeName":"llm-node"}     // 大模型变量
data: {"context":"韩宗辉", "nodeTypeName":"other-node"}           // 全局变量 name
data: {"context":"11", "nodeTypeName":"other-node"}              // 全局变量 age
data: {"context":"22222222222222222222", "nodeTypeName":"output-node"} // 静态文本
```


**判断规则**:
- ✅ 静态文本 → `"output-node"`
- ✅ 变量名以 `llmText` 结尾 → `"llm-node"`
- ✅ 其他变量 → `"other-node"`

#### **(2) 修改 `handleVariables` 方法**
```java
private void handleVariables(String input, String businessKey, String scopeKey, 
        String sessionId, StringBuilder contentBuilder, StringBuilder thinkBuilder, 
        AgentChatMessageEntity agentChatMessageEntity, DelegateExecution delegateExecution, 
        String workflowId, Set<String> syncedLlmVarNames, boolean syncLlmVariables) {
    
    Matcher matcher = VARIABLE_PATTERN.matcher(input);
    String[] parts = input.split("\\$\\{[^}]*}");
    
    int i = 0;
    while (matcher.find()) {
        // 推送静态文本
        if (parts.length > i) {
            String real = parts[i];
            pushReplyContent(..., real, ..., "reply-node");  // ← 静态文本
            contentBuilder.append(real);
        }
        
        // 推送变量
        String expression = matcher.group(0);
        String varName = WorkflowUtil.extractSingleVariableName(expression);
        
        if (StringUtils.isBlank(varName) || !varName.endsWith("llmText")) {
            // 非大模型变量 → other-node
            Object obj = evaluateExpression(delegateExecution, expression, Object.class);
            String realValue = obj == null ? "" : obj.toString();
            pushReplyContent(..., realValue, ..., "other-node");  // ← 非大模型
            contentBuilder.append(realValue);
        } else {
            // 大模型变量 → llm-node
            Object cachedValue = delegateExecution.getVariable(varName);
            String realValue = cachedValue == null ? "" : cachedValue.toString();
            if (StringUtils.isNotBlank(realValue)) {
                pushReplyContent(..., realValue, ..., "llm-node");  // ← 大模型
                contentBuilder.append(realValue);
                syncedLlmVarNames.add(varName);
            }
        }
        i++;
    }
    
    // 推送最后的静态文本
    if (parts.length > i) {
        String real = parts[i];
        pushReplyContent(..., real, ..., "reply-node");
        contentBuilder.append(real);
    }
}
```

// 新增重载方法，接受 nodeTypeName 参数
private void pushOutputChunk(DelegateExecution delegateExecution, 
        String content, boolean isStream, String nodeTypeName) {
    if (StringUtils.isBlank(content)) {
        return;
    }
    if (isStream) {
        push(null, buildChatflowResponse(delegateExecution, content, nodeTypeName), 
             delegateExecution);
        return;
    }
    for (String message : WorkflowUtil.transformStr2List(false, content)) {
        push(null, buildChatflowResponse(delegateExecution, message, nodeTypeName), 
             delegateExecution);
    }
}

// 保留原方法作为兼容

        String content, boolean isStream) {
    pushOutputChunk(delegateExecution, content, isStream, null);
}

private WorkerStreamResponseV1 buildChatflowResponse(
        DelegateExecution delegateExecution, String content, String nodeTypeName) {
    String businessKey = delegateExecution.getBusinessKey();
    String workflowId = delegateExecution.getVariable("sys_workflowId") == null 
        ? businessKey : delegateExecution.getVariable("sys_workflowId").toString();
    String sessionId = delegateExecution.getVariable("sys_sessionId") == null 
        ? "" : delegateExecution.getVariable("sys_sessionId").toString();
    
    // 如果传入了 nodeTypeName，直接使用；否则根据配置判断

    return new WorkerStreamResponseV1(businessKey, workflowId, sessionId, 
        "assistant", "", content, 0, 0, 0, 0.0, null, 
        getCurrentTimeStr(), nodeTypeName);
}

#### **(3) 修改 `streamChatflowOutput` 方法**
```java
private String streamChatflowOutput(DelegateExecution delegateExecution, 
        String template, boolean isStream) {
    Pattern pattern = Pattern.compile("\\$\\{([^}]*)\\}");
    Matcher matcher = pattern.matcher(template);
    String[] parts = template.split("\\$\\{[^}]*}");
    String businessKey = delegateExecution.getBusinessKey();
    StringBuilder finalOutput = new StringBuilder();
    int index = 0;
    
    while (matcher.find()) {
        // 推送静态文本
        if (parts.length > index) {
            String staticPart = parts[index];
            pushOutputChunk(delegateExecution, staticPart, isStream, "output-node"); // ← 静态文本
            finalOutput.append(staticPart);
        }
        
        // 推送变量
        String expression = matcher.group(0);
        String variableName = WorkflowUtil.extractSingleVariableName(expression);
        
        if (StringUtils.isBlank(variableName) || !variableName.endsWith("llmText")) {
            // 非大模型变量 → other-node
            String realValue = "";
            try {
                Object obj = evaluateExpression(delegateExecution, expression, Object.class);
                realValue = obj == null ? "" : obj.toString();
            } catch (Exception e) {
                log.error("输出节点变量解析失败：{}", expression, e);
            }
            pushOutputChunk(delegateExecution, realValue, isStream, "other-node"); // ← 非大模型
            finalOutput.append(realValue);
        } else {
            // 大模型变量 → llm-node
            Object cachedValue = delegateExecution.getVariable(variableName);
            String realValue = cachedValue == null ? "" : cachedValue.toString();
            if (StringUtils.isNotBlank(realValue)) {
                pushOutputChunk(delegateExecution, realValue, isStream, "llm-node"); // ← 大模型
                finalOutput.append(realValue);
            } else {
                LlmStreamPayload payload = consumeLlmStream(...);
                finalOutput.append(payload.content());
            }
        }
        index++;
    }
    
    // 推送最后的静态文本
    if (parts.length > index) {
        String staticPart = parts[index];
        pushOutputChunk(delegateExecution, staticPart, isStream, "output-node"); // ← 静态文本
        finalOutput.append(staticPart);
    }
    
    return finalOutput.toString();
}
```

## 📝 **关键判断逻辑**

### **变量名判断规则**

所有节点类型判断都基于变量名是否以 `"llmText"` 结尾:

```java
// 提取变量名
String varName = WorkflowUtil.extractSingleVariableName(expression);

// 判断类型
if (StringUtils.isBlank(varName)) {
    // 无变量名，视为静态文本
    nodeTypeName = "reply-node"; // 或 "output-node"
} else if (varName.endsWith("llmText")) {
    // 大模型变量
    nodeTypeName = "llm-node";
} else {
    // 其他变量
    nodeTypeName = "other-node";
}
```

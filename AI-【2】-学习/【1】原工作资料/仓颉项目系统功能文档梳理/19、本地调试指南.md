## 19、本地调试指南

---

### **一、环境搭建**

#### **1.1 JDK 安装**

**版本要求**: JDK 21+

**下载地址**:
- 官方：https://adoptium.net/
- 推荐：Eclipse Temurin 21

**安装验证**:
```bash
java -version
# 输出应包含 "version '21.x.x'"

javac -version
# 输出应包含 "javac 21.x.x"
```


---

#### **1.2 PostgreSQL 安装**

**版本要求**: PostgreSQL 12+

**Windows 安装步骤**:

1. **下载安装包**: https://www.postgresql.org/download/windows/
2. **运行安装程序**: 选择安装 PostgreSQL Server
3. **设置密码**: 记住 postgres 用户的密码
4. **端口设置**: 默认 5432
5. **安装 pgAdmin**: 图形化管理工具

**验证安装**:
```bash
# 检查服务状态
pg_ctl status

# 命令行登录
psql -U postgres -p 5432
```


**创建数据库**:
```sql
-- 1. 创建数据库
CREATE DATABASE nlp_agent_test 
WITH OWNER = postgres 
ENCODING = 'UTF8' 
LC_COLLATE = 'Chinese_China.936' 
LC_CTYPE = 'Chinese_China.936' 
TABLESPACE = pg_default 
CONNECTION LIMIT = -1;

-- 2. 创建用户
CREATE USER nlp_agent_test WITH PASSWORD 'your_password';

-- 3. 授权
GRANT ALL PRIVILEGES ON DATABASE nlp_agent_test TO nlp_agent_test;
```


---

#### **1.3 Redis 安装**

**版本要求**: Redis 6.0+

**Windows 安装方式**:

**方式 1: 使用 WSL(推荐)**
```bash
# WSL 中安装
sudo apt update
sudo apt install redis-server
sudo service redis-server start
redis-cli --version
```


**方式 2: 使用 Docker**
```bash
docker run -d --name redis -p 6379:6379 redis:latest
docker ps
```


**方式 3: Windows 原生版本**
- 下载：https://github.com/redisarchive/redis-windows
- 解压后运行 `redis-server.exe`

**验证安装**:
```bash
redis-cli ping
# 返回 PONG 表示成功
```

#### **4.2 Cron 表达式工具**

**在线生成**: https://cron.qqe2.com/

**常用表达式**:
```
# 每分钟执行
*/1 * * * * ?

# 每 5 分钟执行
0 */5 * * * ?

# 每小时执行
0 0 * * * ?

# 每天凌晨 2 点执行
0 0 2 * * ?

# 每周一 8 点执行
0 0 8 ? * MON

# 每月 1 号执行
0 0 0 1 * ?
```

#### **4.3 本地调试定时任务**

**方式 1: 修改 Cron 为高频执行**
```yaml
# 临时修改配置，改为每分钟执行
@Scheduled(cron = "0 */1 * * * ?")
public void statOperationAppData() {
    log.info("开始执行运营数据统计任务");
    // ... 业务逻辑
}
```


**方式 2: 手动触发接口**
```java
// AgentOperationController.java
@PostMapping("/manual/statAppData")
@Operation(summary = "手动触发 app 运营数据统计 (T+1)")
public CommonResult<String> manualStatAppData() {
    try {
        log.info("手动触发 app 运营数据统计，当前时间：{}", LocalDateTime.now());
        agentOperationDashboardService.statOperationAppData();
        log.info("app 运营数据统计完成");
        return CommonResult.success("✅ 统计完成！");
    } catch (Exception e) {
        log.error("手动触发 app 运营数据统计失败:", e);
        return CommonResult.fail("❌ 统计失败：" + e.getMessage());
    }
}
```


**测试步骤**:
1. 启动项目
2. 访问：`POST http://localhost:7002/api/operation/manual/statAppData`
3. 查看日志输出
4. 查询数据库验证：
```sql
-- 查看统计结果
SELECT * FROM agent_operation_app 
ORDER BY stat_date DESC 
LIMIT 10;
```

### **六、常见问题排查**

#### **6.1 项目启动失败**

**问题 1: 数据库连接失败**
```
ERROR: Connection refused
```


**解决方案**:
1. 检查 PostgreSQL 服务是否启动
2. 验证账号密码是否正确
3. 检查端口 5432 是否开放
```bash
# Windows 检查端口
netstat -an | findstr 5432
```


**问题 2: Redis 连接失败**
```
ERROR: Cannot get Jedis connection
```


**解决方案**:
1. 检查 Redis 服务是否启动
2. 验证 Redis 配置
```yaml
spring:
  data:
    redis:
      host: localhost
      port: 6379
```


---

#### **6.2 定时任务未执行**

**问题**: 日志中没有定时任务输出

**排查步骤**:
1. **检查@EnableScheduling 注解**
```java
@SpringBootApplication
@EnableScheduling  // 确保有此注解
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```


2. **检查 Bean 是否被 Spring 管理**
```java
@Service  // 必须加上
public class AgentOperationDashboardServiceImpl {
    @Scheduled(cron = "0 15 0 * * ?")
    public void statOperationAppData() {
        // ...
    }
}
```


3. **查看 Cron 表达式是否正确**
```java
// 错误示例
@Scheduled(cron = "0 15 0 * *")  // 缺少秒和年

// 正确示例
@Scheduled(cron = "0 15 0 * * ?")
```


---

### **八、快速启动脚本**

#### **8.1 Windows 启动脚本**

**start-all.bat**:
```batch
@echo off
echo 启动 PostgreSQL...
net start postgresql-x64-15

echo 启动 Redis...
start redis-server.exe

echo 等待服务启动...
timeout /t 5

echo 启动主服务...
cd agent-builder\agent-build-server
start "Agent Build Server" cmd /k "mvn spring-boot:run -Dspring-boot.run.profiles=dev"

timeout /t 10

echo 启动 Worker 服务...
cd ..\..\agent-worker
start "Agent Worker" cmd /k "mvn spring-boot:run -Dspring-boot.run.profiles=dev"

echo 所有服务已启动！
pause
```


---

#### **8.2 停止脚本**

**stop-all.bat**:
```batch
@echo off
echo 停止所有服务...

echo 停止 PostgreSQL...
net stop postgresql-x64-15

echo 停止 Redis...
taskkill /F /IM redis-server.exe

echo 清理完成！
pause
```
